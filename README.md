### Required Variables
* `BOT_TOKEN`: Create a bot using [@BotFather](https://telegram.dog/BotFather) and get the token.
* `API_ID`: Get this from [my.telegram.org](https://my.telegram.org/apps).
* `API_HASH`: Get this from [my.telegram.org](https://my.telegram.org/apps).
* `MONGO_URI`: Your MongoDB connection string.
* `TMV_URL`: TamilMV website URL (e.g., https://www.1tamilmv.land/).
* `TMV_TORRENT`: Channel ID where torrent files should be uploaded.
* `TMV_LEECH_GRP`: Group ID for leeching purpose.
* `TMV_MIRROR_GRP`: Group ID for mirror links.

### Optional Variables
* `TMV_TORRENT_THUMB`: Custom thumbnail URL for your torrent posts.
* `BOT_TAG`: Your custom brand tag (e.g., вҡЎYourBotNameвҡЎ).

---

### Project Structure
* `bot.py`: Main entry for client and web server.
* `configs.py`: Loads all the above variables.
* `database.py`: Handles MongoDB duplicate tracking.
* `tamilmv.py`: Scraping and upload logic.
